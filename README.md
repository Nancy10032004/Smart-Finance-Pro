# Smart-Finance-Pro

A responsive and user-friendly Personal Finance Management Web Application built using HTML, CSS, and Vanilla JavaScript. The application helps users manage income, expenses, budgets, savings goals, and loan EMI calculations through an interactive dashboard.

## Features

- Dashboard with financial overview
- Income & Expense Tracking
- Add, Edit & Delete Transactions
- Budget Planning
- Savings Goals Tracker
- EMI Calculator
- Interactive Charts & Analytics
- Dark / Light Theme
- LocalStorage Data Persistence
- Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- LocalStorage API
- Lucide Icons
