# Smart Finance Pro

Smart Finance Pro is a personal finance management web application developed using HTML, CSS, and Vanilla JavaScript. The main idea behind this project is to help users manage their daily finances in a simple and organized way without requiring any backend or database.

The application allows users to record income and expenses, create monthly budgets, track savings goals, calculate loan EMIs, and generate monthly financial reports. All data is stored locally in the browser using LocalStorage.

## Features

- Dashboard with balance, income, expenses, and savings overview
- Add, edit, and delete transactions
- Filter and search transaction history
- Category-wise monthly budget management
- Savings goal tracker with progress updates
- EMI calculator with repayment details
- Recurring bill and subscription management
- Financial calendar for upcoming payments
- PDF report generation
- Dark and light theme support
- Import and export data using JSON files

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/SmartFinancePro.git
   ```

2. Open the project folder:
   ```bash
   cd SmartFinancePro
   ```

3. Start a local server using any of the following:
   ```bash
   python -m http.server 8000
   ```
   or
   ```bash
   npx serve
   ```

4. Open your browser and visit:
   ```
   http://localhost:8000
   ```
